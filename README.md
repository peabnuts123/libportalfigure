# libportalfigure

This is a library of data relating to figures from the Skylanders series of video games, such as IDs, and names.

## Disclaimer

This library does not intend to infringe any copyright of the intellectual property holders. No code from any games, promotional materials, or data from NFC tags is contained in this library. All data is purely for reference use only.
